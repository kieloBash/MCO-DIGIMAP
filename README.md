# MCO-DIGIMAP

## RUN THE BACKEND (PYTHON)
1. open terminal
2. cd server

3. (optional)
    .\venv\Scripts\activate 

4.  pip install Flask                         
    pip install opencv-python
    pip install numpy
    pip install Flask-CORS
    pip install imutils
    
5. python server.py


## RUN THE FRONTEND (REACT)
1. open terminal
2. cd client
3. npm run dev
4. localhost:3000
